# 工学部第二部 地域共育実習生 認定研修：現場で通用するエンジニアの「心得」

東京電機大学 工学部第二部 地域共育プロジェクト事務局

## 1. 研修の目的と重要性

### 1.1 なぜ、この研修が必要なのか？

あなたは、単なるアルバイトではありません。大学の推薦を受けた**「地域共育実習生」**という肩書きを持つ、**プロのエンジニアの卵**です。

この研修の合格（認定証の取得）は、企業に対し以下の3点を証明します。

1. **最低限のマナー・意識**を持っている。
    
2. **安全衛生**の基礎知識がある。
    
3. **機密保持**の理解している。
    

> [!CAUTION] 厳格なルール:
> 
> 本プロジェクトの求人に応募できるのは、この研修に合格した認定学生のみです。認定証は、あなたと大学、そして企業の「信頼の証」であることを忘れないでください。

## 2. モジュール 1：安全衛生の鉄則（現場の常識）

技術現場で最も嫌われるのは、「危ない人」です。まず、あなた自身の身の安全と、職場の安全を守るルールを学びます。

### 2.1 現場の基本：5Sの徹底

事故を防ぐ第一歩は、**整理・整頓**です。

1. **整理 (Seiri):** いるものといらないものを分け、いらないものを捨てる。

2. **整頓 (Seiton):** いるものを使いやすい場所に、誰にでもわかるように置く。

3. **清掃 (Seisou):** 職場をきれいにし、ゴミや汚れがない状態を維持する。

4. **清潔 (Seiketsu):** 3S（整理・整頓・清掃）を維持する。

5. **躾 (Shitsuke):** 決められたルールや手順を正しく守る習慣をつける。

### 2.2 工具・道具の取り扱い

製造現場では、工具の管理があなたの「仕事の姿勢」を示します。

1. **使ったら元の場所に戻す:**

    - 工具は**必ず元の場所に、元の向きで**戻すこと。
    - 「あとでまとめて片付ける」は厳禁。使い終わったら**すぐに**戻す。
    - 工具を探す時間 = 生産性のロス。あなたの後片付けが現場全体の効率を左右します。

2. **借りた工具は報告:**

    - 他の場所から工具を借りた場合は、**誰に借りたか、いつ返すか**を明確に伝えること。
    - 無断で持ち出すと、次に使う人が困ります。

3. **破損・紛失は即報告:**

    - 工具を壊したり、なくしたりした場合は、**隠さずすぐに報告**すること。
    - 黙っていると、次の作業者が困り、事故につながる可能性もあります。

4. **退勤前の確認:**

    - 帰る前に、自分が使った工具・道具が**すべて元の場所にあるか**を確認すること。
    - これができる学生は、現場で「任せられる人」として評価されます。
    

### 2.3 危険予知（KYT）と安全装備

現場での事故のほとんどは「だろう運転」と「予測不足」から起こります。

1. **KYT (危険予知訓練):**
    
    - 作業を始める前に、「**ここにどんな危険が潜んでいるか**」を予測し、指差し・声出し確認を必ず行ってください。
        
2. **ヒヤリハット報告:**
    
    - 「危なかった」「もう少しで怪我をするところだった」という出来事（ヒヤリハット）は、**絶対に隠さず**すぐに管理者に報告してください。あなたの報告が、次の大事故を防ぎます。
        
3. **スマホ禁止:**
    
    - 機械の稼働中や、工具・刃物の近くでは、絶対にスマホに触らないこと。
        

## 3. モジュール 2：プロの報・連・相

現場はスピードと正確性を求めます。学生気分を捨て、プロとしてのコミュニケーションを心がけてください。

### 3.1 指示の受け方（製造現場の鉄則）

製造現場では、指示を正確に理解することが品質と安全の基本です。

1. **必ずメモを取る:**

    - 指示を受けたら、**手元のメモ帳にすぐ書く**こと。「覚えました」は信用されません。
    - メモすべき内容：**何を、いくつ、いつまでに、どの順番で、どこに置くか**
    - ポケットに入る小さなメモ帳とペンを常に携帯してください。

2. **復唱確認:**

    - 指示を受けたら、**必ず復唱**して確認すること。
    - 例：「〇〇を△個、□□時までに、この棚に置く。以上でよろしいでしょうか？」
    - 復唱することで、聞き間違いや勘違いを防げます。

3. **わからないことは「その場で」聞く:**

    - 作業を始めてから「やっぱりわかりません」は最悪です。
    - **指示を受けた直後**に不明点を質問すること。後から聞きに行くのは二度手間です。

### 3.2 現場で試される「報告・連絡・相談」

- **✅ 「わからない」は勇気:**

    - 知ったかぶりをして機械を触り、機材を破損させたり、怪我をしたりするのが最悪です。**「これは、どうすればいいでしょうか？」**と、作業前に質問できる学生こそが評価されます。

- **🚨 遅刻・欠席は「電話」で:**

    - メールやLINEではなく、**必ず電話**で担当者に連絡してください。企業は、あなたの出勤を前提に生産計画を立てています。「1分でも遅れる」とわかった時点で電話を。

### 3.3 作業終了時の報告

作業が終わったら、**必ず指示者に報告**すること。「終わりました」の一言が信頼を築きます。

1. **完了報告:**

    - 「〇〇の作業、完了しました。次は何をすればよろしいでしょうか？」
    - 勝手に次の作業に移らない。指示を待つこと。

2. **異常・ミスの即時報告:**

    - 失敗やミスは**隠さず、すぐに報告**すること。
    - 報告が遅れるほど、被害は拡大し、信頼は失われます。
    - 例：「申し訳ありません。〇〇を△個、不良にしてしまいました」
    - **正直に報告できる人こそ、現場で信頼されます。**
        

## 4. モジュール 3：ビジネスマナーの基礎（信頼を築く）

企業は、あなたの技術力だけでなく、「人として信頼できるか」を見ています。マナーは、スムーズな業務遂行のための潤滑油です。

### 4.1 敬語と正しい言葉遣い

|**場面**|**正しい言葉**|**間違いやすい言葉**|
|---|---|---|
|相手に聞くとき|**恐れ入りますが**、〜|すみませんが、〜|
|理解したとき|**承知いたしました** / かしこまりました|わかりました / 了解です / りょ|
|相手を呼ぶとき|**〇〇様** / 〇〇部長|〇〇さん / 〇〇君|
|自分のことを言うとき|**私** / わたくし|俺 / 自分 / うち|
|お願いするとき|**お手数ですが** / 恐れ入りますが|ちょっといいですか|
|謝るとき|**申し訳ございません**|すみません / ごめんなさい|
|感謝するとき|**ありがとうございます**|どうも / あざす|
|確認するとき|**よろしいでしょうか**|いいですか|
|席を外すとき|**少々お待ちください**|ちょっと待ってください|
|物を受け取るとき|**頂戴いたします**|もらいます|
|知らないとき|**存じ上げません** / 勉強不足で申し訳ありません|知りません / わかりません|
|出勤時|**おはようございます**|おはよう / ういっす|
|帰るとき|**お先に失礼いたします**|お先です / お疲れ様です（目上に対しては不適切）|
|指示を受けたとき|**かしこまりました**|了解です / りょ / OK|

### 4.2 身だしなみと態度

- **清潔感:** 服装（指定がある場合）・髪型・爪は清潔に保ってください。現場では、安全のため長い髪はまとめ、アクセサリーは外すことが基本です。
    
- **挨拶:** 笑顔で、相手の目を見て立ち止まって挨拶をしてください。「おはようございます」「お先に失礼します」は基本中の基本です。
    
- **訪問・退社時のマナー:** - 訪問先（企業）では、**名刺交換**の機会があるかもしれません。相手から受け取る際は両手で受け取り、すぐにしまわず机の上に置いてください。
    
    - 帰るときは、**「席を立つ前に」**上長に声をかけ、**「今日の業務内容と明日の予定」**を報告してから退社してください。

## 6. モジュール 5：現場の禁止事項（絶対にやってはいけないこと）

以下の行為は、**認定取消**や**損害賠償請求**の対象となる重大な違反です。

### 6.1 機械・設備の無断操作

- **許可なく機械に触らない:**

    - 「これくらい大丈夫だろう」は絶対にNG。
    - 機械の操作は、**必ず指導者の許可を得てから**行うこと。
    - 無断操作による事故は、あなたの責任となります。

- **「見て覚える」の前に「聞いて確認」:**

    - 先輩がやっているのを見たからといって、勝手に真似しないこと。
    - 必ず「私もやってよろしいですか？」と確認を取ること。

### 6.2 SNS・写真撮影の禁止

- **職場内の写真・動画撮影は厳禁:**

    - 製品、図面、設備、作業風景など、**一切の撮影禁止**。
    - 「自分用のメモ」であっても許可なく撮影してはいけません。

- **SNSへの投稿禁止:**

    - 会社名、業務内容、製品情報などを、SNS（X、Instagram、TikTok等）に投稿することは**機密漏洩**にあたります。
    - 「バイト先の工場なう」のような軽い投稿も厳禁です。
    - 違反した場合、**認定取消・損害賠償**の対象となります。

### 6.3 無断欠勤・バックレの禁止

- **「バックレ」は絶対禁止:**

    - 連絡なしに来なくなる行為（バックレ）は、社会人として最も信頼を失う行為です。
    - 企業は生産計画をあなたの出勤に合わせて組んでいます。
    - 無断欠勤は、**認定の即時取消**となります。

- **辞めたい場合は正式に申し出る:**

    - 辞めたくなった場合は、**できるだけ早めに**担当者に相談すること。
    - 「合わないから」「つまらないから」で逃げるのではなく、きちんと理由を伝え、引継ぎを行うこと。
    - **正しく辞められる人は、次の就職でも評価されます。**

### 6.4 ベテラン・年上への敬意

- **年齢や経験に関係なく敬意を払う:**

    - 現場には、何十年もその仕事に携わってきたベテランがいます。
    - 学歴や知識ではなく、**現場での経験と技術**を尊重すること。
    - 「そんなやり方は古い」などの発言は厳禁。まず学ぶ姿勢を見せてください。

### 6.5 もしもの時の相談窓口（あなたの安全を守るために）

職場でハラスメントを受けたり、賃金が支払われないなど、研修や利用規約に反する重大なトラブルに巻き込まれてしまった場合は、**一人で抱え込まず、すぐに大学の学生厚生担当に連絡してください。**

*   **電話番号:** XX-XXXX-XXXX
*   **メール:** student-welfare@dendai.ac.jp

あなたの秘密は厳守されます。大学が責任を持って対応し、必要に応じて[10_トリアージ規定](10_トリアージ規定.md)に基づき企業への措置を講じます。

## 7. モジュール 6：成長のためのPDCAサイクル

実習を単なる労働で終わらせず、キャリアに活かすためには、意識的な成長プロセスが必要です。

### 7.1 実習を「学び」に変える PDCA

1. **P (Plan - 計画):**
    
    - **目標設定:** 今週、この業務を通して「何を身につけたいか（例：CADソフトの特定機能、図面読解力）」を明確にしてください。
        
2. **D (Do - 実行):**
    
    - 積極的に作業を行い、報・連・相を徹底する。
        
3. **C (Check - 評価):**
    
    - **反省:** 今週の目標は達成できたか？できなかった場合、原因は何か？（例：知識不足、時間が足りなかった）を記録してください。
        
4. **A (Action - 改善):**
    
    - **次の行動:** 原因に基づき、来週の「Plan（計画）」を修正してください。この改善こそが、あなたの成長スピードを加速させます。
        

> [!TIP] キャリア価値の最大化
> 
> 実習終了時には、企業に「[実務経験証明書](32_実務経験証明書_雛形.md)（大学指定テンプレート）」への記入・押印を依頼してください。このPDCAの記録は、**証明書に記載する「業務実績レポート」**の強力な裏付けとなります。

## 8. 認定証の自動取得フロー

### 8.1 応募資格の取得方法

1. **この研修動画（または資料）を最後まで視聴（閲覧）する。**
    
2. **認定テストに合格する**（合格点 70点以上）。
    
3. **合格後、あなたのアドレスに「合格通知メール」が自動送信されます**。
    
    - このメールが**「地域共育実習生 認定証」**となります。
        
4. **メール内のURL/パスワード**から、企業リスト（求人マスター）を閲覧し、企業に直接応募する。
