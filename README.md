# 地域共育プログラム プロジェクト資料

このディレクトリには、工学部第二部が主導する「地域共育プログラム」プロジェクトに関連するすべての定義書、規約、マニュアル、およびテンプレートが含まれています。

## ファイル構成と命名規則

プロジェクトのドキュメントは、対象読者と内容に基づいてカテゴリ分けされ、ファイル名の先頭に数字の接頭辞が付与されています。これにより、論理的な順序でファイルを閲覧し、管理することができます。

各ファイルはObsidianなどのMarkdownエディタでの利用を想定しており、内部リンクによって関連情報へ容易にアクセスできます。

### カテゴリ分類

- **`00_`**: プロジェクト全体の定義と、ウェブサイトやフォームの要件定義。
- **`10_`**: 管理者向けの内部規定や運用マニュアル。
- **`30_`**: 学生向けの資料（研修、規約、テンプレート）。
- **`40_`**: 企業向けの資料（案内、規約）。

## サイトのビルド方法

このプロジェクトのドキュメントは、`site/`ディレクトリに静的なHTMLウェブサイトとして出力できます。サイトの生成にはPythonを使用します。

### 1. 準備

ビルドスクリプトを実行する前に、必要なライブラリをインストールする必要があります。

```bash
pip install -r requirements.txt
```
(注: `pip`の代わりに`pip3`が必要な場合があります。また、仮想環境の使用を推奨します。)

### 2. ビルド実行

以下のコマンドを実行すると、ルートディレクトリにある`.md`ファイルがHTMLに変換され、`site/`ディレクトリ内に再生成されます。

```bash
python build.py
```
(注: `python`の代わりに`python3`が必要な場合があります。)

`site`ディレクトリをウェブサーバーでホストすることで、完成したウェブサイトを閲覧できます。
