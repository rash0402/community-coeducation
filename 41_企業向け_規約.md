# 東京電機大学 工学部第二部 地域共育プログラム 利用規約

本規約は、東京電機大学（以下「本学」といいます）が運営する「工学部第二部 地域共育プログラム」（以下「本サービス」といいます）を利用して求人情報を掲載する企業・団体（以下「利用企業」といいます）に適用されます。本サービスへの利用申し込み（求人登録）を行った時点で、本規約のすべての条項に同意したものとみなします。

## 【本規約におけるトラブル対応レベル（補足）】

本学が定めるトラブル対応の深刻度レベルと、本規約の条文の対応関係は以下の通りです。本レベルは、学生および事務局がトラブルを分類・対応するための内部基準ですが、利用企業もこのレベルで運用されることをご確認ください。

|   |   |   |
|---|---|---|
|**対応レベル**|**事象の性質**|**規約上の対応条文**|
|**Lv.1（日常トラブル）**|遅刻、欠席、業務ミスなど（企業の就業規則で対応可能な範囲）|第2条3項（本学の非介入）、第10条1項（免責事項）|
|**Lv.2（継続的・軽微な違反）**|勤務態度不良の再三の報告、掲載基準（移動時間など）の再三の不適合|第8条1項（掲載停止処分）、第9条（再開プロセス）|
|**Lv.3（緊急・重大な違反）**|ハラスメント、労災保険の不適用、賃金未払い、重大な契約違反|第8条2項（永久抹消処分）|

## 第1条（目的）

本サービスは、本学工学部第二部（夜間部）に在籍する学生に対し、学業と両立可能な技術系就労の機会（アルバイト・有給インターンシップ等）に関する情報を提供し、地域企業の技術継承および学生のキャリア形成を支援することを目的とします。

## 第2条（サービスの性質・本学の役割）

1. **【プラットフォームの提供】** 本サービスは、利用企業から提供された求人情報を学生に閲覧させるための「情報の場」を提供するものです。

2. **【非仲介】** 本学は、職業安定法に基づく職業紹介（斡旋）は行いません。面接日程の調整、採用の合否決定、雇用条件の交渉等は、すべて利用企業と応募学生との間で直接行っていただきます。

3. **【契約の主体】** 雇用契約は、利用企業と学生との間で直接締結されるものとし、本学は雇用契約の当事者にはなりません。

4. **【本学が行うこと】** 本学は、本サービスにおいて以下のサポートを提供します。
    - 学生への事前研修の実施（安全衛生・ビジネスマナー・コンプライアンス）
    - 求人情報の認定学生への限定公開
    - トラブル報告フォームの設置および受理
    - 重大な規律違反があった学生に対する認定取消処分
    - 緊急時の相談窓口の設置（詳細は「[42_企業向けガイド](42_企業向けガイド.md)」参照）

5. **【本学が行わないこと】** 本学は、以下については対応いたしません。
    - 日常的なシフト調整・欠勤対応への介入
    - 面接日程の調整や条件交渉の代行
    - 採用のマッチング保証
    - 雇用契約内容への介入・仲裁
    - 労災手続きの代行
    - 電話による即時対応（報告フォームでの受付が基本となります）
    

## 第3条（利用資格）

本サービスを利用できるのは、以下の要件をすべて満たす企業・団体に限ります。

1. 法人格を有し、実在する事業所を持っていること。
    
2. 労働基準法、労働安全衛生法、最低賃金法、その他関係法令を遵守していること。
    
3. 本学工学部第二部の教育方針（働きながら学ぶ）を理解し、学生の学業優先に配慮できること。
    
4. 風俗営業等の規制及び業務の適正化等に関する法律に定める風俗営業、性風俗関連特殊営業を行っていないこと。
    

## 第4条（求人情報の掲載基準）

以下のいずれかに該当する求人は、掲載をお断り（または削除）します。

1. **【学業支障】** 勤務地から本学（東京千住キャンパス）までの移動時間を考慮すると、18:00からの講義開始に間に合わないもの。
    
2. **【業務内容】** 工学的知識・技術の習得に寄与しない業務（単なる清掃、ビラ配り、ホールスタッフ等）のみを目的とするもの。
    
3. **【虚偽表示】** 実際の労働条件と異なる好条件を提示する「おとり求人」。
    
4. **【公序良俗】** 公序良俗に反する、または本学の品位を損なうおそれのあるもの。
    

## 第5条（雇用契約と学業への配慮・保険）

利用企業は、本学学生を雇用するにあたり、以下の事項を遵守するものとします。

1. **【書面の交付】** 労働基準法第15条に基づき、必ず労働条件通知書（雇用契約書）を作成・交付すること。
    
2. **【学業優先】** 18:00からの講義に出席できるよう、残業を命じないこと。また、大学の定期試験期間（7月・1月等）における休暇取得またはシフト調整に最大限配慮すること。
    
3. **【契約期間】** 大学の学期変更（時間割変更）に対応するため、雇用契約期間は原則として「当該学期の末日（3月31日または9月30日）」までとし、更新時に次学期のシフトを再調整することを推奨します。
    
4. **【安全配慮・保険】** 学生の安全確保に万全を期し、危険な業務や未熟練の状態で高度な操作を行わせないこと。また、**労働者災害補償保険（労災保険）を必ず適用すること**。
    
5. **【賠償責任】** 学生が業務中に第三者または利用企業の設備等に損害を与えた場合、故意または重大な過失がある場合を除き、**原則として利用企業がその責任を負うものとします（使用者責任）**。本学の学生保険（学研災等）は適用されないことを予め了承すること。
    

## 第6条（掲載期間と更新）

1. 求人情報の掲載期間は、原則として「当該学期の末日」までとします。
    
2. 次学期以降も掲載を希望する場合は、本学が定める所定の期間内に更新手続きを行うものとします。手続きがない場合、掲載は自動的に終了します。
    

## 第7条（禁止事項）

利用企業による以下の行為を禁止します。

1. 採用した学生に対し、求人票と著しく異なる条件での労働を強要すること。
    
2. 学生に対し、違法行為、ハラスメント（セクシャルハラスメント、パワーハラスメント等）、差別的言動を行うこと。
    
3. 本サービスを通じて取得した学生の個人情報を、採用活動以外の目的（自社商品の勧誘、名簿業者への転売等）に利用すること。
    
4. 学生に対し、「大学の紹介だから辞められない」等の誤った説明を行い、退職を妨害すること。
    

## 第8条（利用の停止・登録抹消）

1. **【掲載停止（Lv.2処分）】** 本学は、利用企業が第4条（掲載基準）に定める基準を再三満たさないと判断した場合、または本規約の軽微な条項に違反した場合、当該企業の求人掲載を**1学期間（6ヶ月間）停止**し、以後の利用を制限することができるものとします。
    
2. **【永久抹消（Lv.3処分）** 本学は、以下のいずれかに該当すると判断した場合、事前の通知なく求人情報の掲載を停止し、**以後の利用を永久にお断り（登録抹消）**することができるものとします。
    
    - 第7条2項（ハラスメント、違法行為）に該当する行為が確認された場合。
        
    - 第5条4項（労災保険の適用）に違反し、学生の安全・保険を確保しなかった場合。
        
    - 賃金未払いなど、重大な労働基準法違反が確認された場合。
        
    - その他、本学が提携企業として著しく不適当と判断した場合。
        

## 第9条（掲載再開のプロセス）

1. **【再開の申請】** 前条1項（掲載停止：Lv.2処分）により処分を受けた利用企業は、停止期間終了後に限り、本学所定の再開手続きを行うことで、掲載再開を申請できるものとします。
    
2. **【再開の条件】** 再開申請時、利用企業は、違反事由に対する**「改善計画書」**を本学に提出し、事務局との**面談**を経て、再登録の可否について承認を得るものとします。承認が得られない場合、掲載は再開されません。
    
3. **【永久抹消の不適用】** 前条2項（永久抹消：Lv.3処分）に該当する利用企業については、本条の再開プロセスは適用されません。
    

## 第10条（免責事項）

1. **【トラブルの解決】** 利用企業と学生との間で生じた一切のトラブル（採用選考、労働条件、事故、損害賠償等）について、本学は法的責任を負わず、当事者間で解決するものとします。
    
2. **【採用の非保証】** 本サービスは、利用企業に対する学生の応募や採用を保証するものではありません。応募がなかったことによる損害について、本学は一切の責任を負いません。
    
3. **【サービスの中断】** 本学は、システムの保守点検や不可抗力（天災、停電等）により、予告なく本サービスを中断・停止することがあります。これによる損害について、本学は責任を負いません。
    

## 第11条（反社会的勢力の排除）

利用企業は、自らが暴力団、暴力団員、暴力団関係企業、総会屋、社会運動等標ぼうゴロ、特殊知能暴力集団その他これらに準ずる者（以下「反社会的勢力」といいます）に該当しないこと、および反社会的勢力と関わりを持たないことを表明・保証するものとします。

## 第12条（準拠法・管轄裁判所）

本規約の解釈にあたっては、日本法を準拠法とします。本サービスに関して紛争が生じた場合には、東京地方裁判所を第一審の専属的合意管轄裁判所とします。

附則

本規約は、2025年11月29日から施行します。
