# ペルソナ分析手法

本ドキュメントでは、[ペルソナ定義](PERSONA.md)で定義した14名のペルソナを活用し、施策検討やリスク評価を行うための分析手法を定義する。

---

## 1. マトリクス分析

### 1.1 目的
施策や機能に対する各ペルソナの反応を一覧化し、**賛成・反対の分布**と**潜在的な問題点**を可視化する。

### 1.2 手順

1. 検討対象の施策を明確に定義する
2. 全ペルソナ（または関連するペルソナ）の反応を予測
3. マトリクスに記入し、傾向を分析

### 1.3 テンプレート

**集計:**
- 賛成: __ 名

**判定基準:**
- 反対が3名以上 → 施策の修正または追加対策を検討
- 特定カテゴリ（学生/企業等）で反対多数 → そのカテゴリ向けの補完策を検討

### 1.4 記入例

**施策:** 「認定研修の動画視聴を必須化（30分）」

| ペルソナ | ID | 賛成 | 中立 | 反対 | 理由・懸念 |
|----------|-----|:----:|:----:|:----:|------------|
| 経済自立型 | A1 | | | ○ | 時間がもったいない |
| キャリア志向型 | A2 | ○ | | | スキル証明になる |
| 学業優先型 | A3 | | ○ | | 短ければ許容 |
| 模索中型 | A4 | | | ○ | 面倒くさい |
| 積極投資型 | B1 | ○ | | | 質の担保になる |
| 様子見型 | B2 | ○ | | | 安心材料になる |
| 即戦力志向型 | B3 | | ○ | | 効果は懐疑的 |
| 育成熱心型 | C1 | ○ | | | 基礎があると助かる |
| 多忙型 | C2 | ○ | | | 教える手間が減る |
| 懐疑型 | C3 | | ○ | | どうせ見ても変わらない |
| 事務効率型 | D1 | ○ | | | 自動化できる |
| 学生保護型 | D2 | ○ | | | 安全教育は必須 |
| 地域活性推進型 | E1 | ○ | | | 質の担保は重要 |

## 2. 重み付けスコアリング

### 2.1 目的
ペルソナごとの**重要度**や**影響度**を数値化し、施策の優先順位付けや意思決定を定量的に行う。

### 2.2 重み付け基準

各ペルソナに「影響度」の重みを設定する。

| ペルソナ | ID | 重み | 根拠 |
|----------|-----|:----:|------|
| **学生** |
| 経済自立型 | A1 | 3 | 主要ターゲット層 |
| キャリア志向型 | A2 | 3 | 主要ターゲット層 |
| 学業優先型 | A3 | 2 | 参加可能性は中程度 |
| 模索中型 | A4 | 2 | 参加可能性は中程度 |
| **経営者** |
| 積極投資型 | B1 | 3 | 初期採用企業として重要 |
| 様子見型 | B2 | 3 | 最も多い層、説得が必要 |
| 即戦力志向型 | B3 | 1 | マッチしにくい |
| **現場担当** |
| 育成熱心型 | C1 | 2 | 成功の鍵を握る |
| 多忙型 | C2 | 2 | 現実的に多い層 |
| 懐疑型 | C3 | 1 | 説得コストが高い |
| **大学** |
| 事務効率型 | D1 | 3 | 運用の持続性に直結 |
| 学生保護型 | D2 | 3 | リスク管理に必須 |
| **商工会議所** |
| 地域活性推進型 | E1 | 2 | 広報・拡大に重要 |
| 会員サービス型 | E2 | 1 | 間接的な関与 |

**重み合計:** 31

### 2.3 スコアリング方法

1. 各ペルソナの反応を点数化
   - 強く賛成: +2
   - 賛成: +1
   - 中立: 0
   - 反対: -1
   - 強く反対: -2

2. 重み × 反応スコア を計算

3. 合計スコアで判定

**判定基準:**
| 合計スコア | 判定 |
|:----------:|------|
| +20以上 | 強く推奨 |
| +10〜+19 | 推奨（軽微な調整で実施） |
| 0〜+9 | 要検討（反対層への対策必要） |
| -1〜-10 | 慎重（大幅な修正が必要） |
| -11以下 | 見送り（根本的な見直し） |

## 3. 対立ペア討論

### 3.1 目的
対立する利害を持つペルソナ同士で**仮想討論**を行い、妥協点や新たな解決策を発見する。

### 3.2 主要な対立ペア

| 対立軸 | ペルソナA | ペルソナB | 論点 |
|--------|-----------|-----------|------|
| 収入 vs 成長 | 経済自立型 A1 | キャリア志向型 A2 | 時給と経験のバランス |
| 育成 vs 即戦力 | 積極投資型 B1 | 即戦力志向型 B3 | 研修の深さ・長さ |
| 教えたい vs 期待しない | 育成熱心型 C1 | 懐疑型 C3 | 学生受入れの是非 |
| 効率 vs 安全 | 事務効率型 D1 | 学生保護型 D2 | 手続き・チェックの量 |
| 学生の希望 vs 企業の都合 | 学生 A1-A4 | 経営者 B3 | 労働条件の設定 |
| 推進 vs 慎重 | 地域活性推進型 E1 | 会員サービス型 E2 | 商工会議所の関与度 |

### 3.4 討論例

**議題:** 「研修時間を2時間に延長すべきか？」

**ペルソナA:** 学生保護型 D2
**ペルソナB:** 経済自立型 A1

---

**Round 1: 各自の主張**

【D2の主張】
安全衛生教育は2時間でも足りないくらい。現場で怪我をしてからでは遅い。学生を守るためには、しっかりとした研修が必要。

【A1の主張】
2時間も拘束されるのは厳しい。その時間があれば働いて稼ぎたい。最低限のことだけ教えてくれればいい。

---

**Round 2: 相手への反論・質問**

【D2からA1へ】
怪我をしたら働けなくなる。長期的に見れば2時間の投資は安いのでは？

【A1からD2へ】
動画で済む内容もあるのでは？全部対面でやる必要があるのか？

---

**Round 3: 妥協点の模索**

【D2が譲れる点】
動画視聴（自宅可）と対面のハイブリッドは検討可能

【A1が譲れる点】
安全に関わる部分は真剣に受ける

【D2が譲れない点】
安全衛生の基本内容は省略不可

【A1が譲れない点】
拘束時間は1時間以内にしてほしい

---

**結論**

【合意できた点】
- 動画視聴（30分・自宅可）＋対面確認テスト（30分）のハイブリッド形式

【合意できなかった点】
- 対面部分の完全廃止

【提案される解決策】
研修を「オンライン動画30分（自宅）」＋「対面30分（大学）」の計1時間構成に変更

## 4. 活用場面

| 場面 | 推奨手法 | 理由 |
|------|----------|------|
| 新機能・施策の検討 | マトリクス分析 | 全体の賛否を俯瞰できる |
| 複数案の優先順位付け | 重み付けスコアリング | 定量的に比較できる |
| 対立が予想される施策 | 対立ペア討論 | 妥協点を事前に発見できる |
| リスク評価 | マトリクス＋対立ペア | 反対層の懸念を深掘りできる |
| 規約・ルールの改定 | 全手法併用 | 多角的な検証が必要 |
