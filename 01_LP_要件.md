# 地域共育プログラム LPサイト要件定義書（Phase 0）

> [!NOTE] Phase 0 の方針
>
> **スモールスタート × 最小運用**を優先し、以下を徹底する：
> - モデル企業 **5社以下**、学生 **10〜20名** を想定
> - 事務局の月間工数 **2時間以下**
> - 自動化ゼロ・ツール最小・人力で回せる規模

## 1. サイト概要

|   |   |
|---|---|
|**項目**|**定義**|
|**サイト数**|**2ページ**（企業向け / 学生向け）|
|**技術構成**|静的HTML + CSS（GitHub Pages でホスティング）|
|**デザイン**|シンプル・モバイルファースト・1カラム構成|
|**目標**|最小限のコンテンツで「信頼」と「行動」を促す|

## 2. LP-A：企業向け

### 2.1 基本情報

|   |   |
|---|---|
|**ターゲット**|五十嵐先生のリレーションで声掛けするモデル企業（5社程度）|
|**目標**|プロジェクト概要の理解 → 参加意思の確認|
|**CTA**|「参加を希望する」→ 事務局メールへ連絡（Phase 0 ではフォーム不要）|

| **ヘッダー** | 東京電機大学 工学部第二部 地域共育プログラム |
| **キャッチコピー** | 大学公認の「技術を学ぶ学生」と出会える。費用ゼロ。 |
| **3つのメリット** | ① 掲載無料 ② 事前研修済みの学生 ③ 学期単位の柔軟な契約 |
| **参加条件** | 18時退社の保証 / 労災保険の適用 / 試験期間の配慮 |
| **参加の流れ** | お問い合わせ → ヒアリング → 求人掲載 → 学生から直接応募 |
| **CTA** | 「参加についてのお問い合わせはこちら」→ メールリンク |
| **フッター** | 東京電機大学 工学部第二部 / 問い合わせ先 |

## 3. LP-B：学生向け

### 3.1 基本情報

|   |   |
|---|---|
|**ターゲット**|工学部第二部在籍の学生（経済的自立 + キャリア形成を目指す層）|
|**目標**|認定研修（PDF）の閲覧 → テスト合格 → 求人リスト閲覧|
|**CTA**|「認定研修を受けて応募資格を得る」→ 研修PDF + テストへ誘導|

### 3.2 コンテンツ構成

| セクション | 内容 |
|-----------|------|
| **ヘッダー** | 東京電機大学 工学部第二部 地域共育プログラム |
| **キャッチコピー** | 昼はエンジニア、夜は学生。履歴書に書ける「公認の実務経験」を手に入れよう。 |
| **コンビニバイトとの違い** | 時給だけじゃない。「技術が身につく」「就活で使える」 |
| **3つの保証** | ① 学業優先（18時の講義に間に合う退勤・試験休み保証） ② 安全保証（労災適用） ③ キャリア価値（公認実習生の肩書き） |
| **参加の流れ** | 研修資料を読む（10分）→ テストに合格 → 求人リストを見る → 企業に直接応募 |
| **Q&A** | よくある質問3〜5件 |
| **CTA** | 「認定研修を受ける」→ 研修ページ or Google Forms テストへ |
| **フッター** | 東京電機大学 工学部第二部 / 問い合わせ先 |

### 3.3 Q&A（掲載内容）

| 質問 | 回答 |
|------|------|
| **成績が悪くても参加できますか？** | 退学勧告等を受けていなければ参加可能です。 |
| **危険な作業はありますか？** | 参加企業は労災保険の適用が義務付けられています。 |
| **試験期間中も働く必要がありますか？** | いいえ。試験期間の休暇は企業の義務です。 |
| **途中で辞められますか？** | 学期末での契約終了を推奨しています。 |

## 4. デザインガイドライン（簡略版）

### 4.1 カラー

| 用途 | HEXコード |
|------|-----------|
| **プライマリ（信頼）** | `#1E3A5F` |
| **アクセント（CTA）** | `#E76F51` |
| **背景** | `#FFFFFF` / `#F8F9FA` |
| **テキスト** | `#333333` |

### 4.2 フォント

- **日本語**: Noto Sans JP（Google Fonts）
- **サイズ**: 本文 16px / 見出し 24〜32px

### 4.3 レイアウト

- **モバイルファースト**（横幅 max-width: 800px）
- **1カラム構成**（シンプルに上から下へ）
- **CTAボタン**: 背景 `#E76F51`、白文字、角丸 8px

## 5. 技術仕様

|   |   |
|---|---|
|**ホスティング**|GitHub Pages（無料・HTTPS対応）|
|**ファイル構成**|`index.html`（学生向け）、`business.html`（企業向け）、`style.css`（共通）|
|**外部依存**|Google Fonts（Noto Sans JP）のみ|
|**JavaScript**|不要（静的HTMLのみ）|
|**Analytics**|Phase 0 では不要（Phase 1 で導入検討）|

## 6. Phase 0 で「やらないこと」

| 項目 | 理由 |
|------|------|
| Google Forms での求人登録 | モデル企業は直接ヒアリングで十分 |
| 認定研修動画 | PDF資料 + 簡易テストで代替 |
| 求人リストの自動連携 | 手動でスプレッドシート共有 |
| Google Analytics | 初期は計測不要 |
| Cookie同意バナー | Analytics 不使用のため不要 |
| 企業の声セクション | Phase 1 以降で追加 |
