# 企業向けガイド：学生との協働を成功させるために

本ガイドは、地域共育プログラムに参加いただく企業様向けの実務マニュアルです。法的事項については「[41_企業向け利用規約](41_企業向け利用規約.md)」をご確認ください。

## 1. トラブル発生時の対応フローチャート

何か問題が起きた際は、以下のフローに従って対応をお願いします。

### 1.1 学生が無断欠勤した場合

```
学生が連絡なく出勤しない
        ↓
【企業】学生本人に電話連絡（1日目）
        ↓
    連絡がつかない
        ↓
【企業】翌日も電話連絡（2日目）
        ↓
    連絡がつかない
        ↓
【企業】3日目も連絡がつかない場合
        ↓
【企業】大学にトラブル報告フォームを送信
        ↓
【大学】事実確認・認定取消処理
        ↓
【大学】企業に結果を通知
```

> [!NOTE]
> 1〜2日の欠勤は企業内で対応をお願いします。3日以上連絡が取れない場合のみ大学にご報告ください。

### 1.2 学生が怪我をした場合（労災）

```
学生が業務中に怪我をした
        ↓
【企業】応急処置・医療機関への搬送
        ↓
【企業】労災保険の手続き（企業の義務）
        ↓
【任意】大学に報告フォームで連絡
       （重大事故の場合のみ）
```

> [!IMPORTANT]
> 労災保険の手続きは企業様の責任で行っていただきます。大学は労災手続きの代行は行いません。

### 1.3 学生の勤務態度に問題がある場合

```
勤務態度に問題がある（遅刻常習、指示無視など）
        ↓
【企業】学生本人に直接注意・指導
        ↓
    改善しない
        ↓
【企業】口頭または書面で警告
        ↓
    それでも改善しない
        ↓
【企業】大学にトラブル報告フォームを送信
        ↓
【大学】事実確認・認定取消の検討
        ↓
【大学】企業に結果を通知
```

> [!NOTE]
> まずは企業内での指導をお願いします。指導しても改善しない場合にご報告ください。

### 1.4 ハラスメント・重大な違反が発覚した場合

```
学生によるハラスメント・機密漏洩・重大な規律違反
        ↓
【企業】証拠の保全（メール、写真など）
        ↓
【企業】大学にトラブル報告フォームを送信
        ↓
【大学】事実確認（必要に応じて聞き取り）
        ↓
【大学】認定取消処分・企業への通知
```

> [!CAUTION]
> 重大な違反については、速やかにご報告ください。認定取消処分を検討します。

## 2. 学生の認定確認方法

面接や採用時に、学生が本プログラムの認定研修を修了しているかを**オンラインで確認**できます。

### 2.1 認定IDとは

学生は研修合格時に、以下の形式の**認定ID**を取得しています。

```
TDU-2025S-00001
```

このIDは学生のスマートフォンに保存された合格メールに記載されています。

### 2.2 認定確認の手順

```
【面接時の確認フロー】

学生に「認定IDを教えてください」と依頼
        ↓
学生がスマホの合格メールを提示、またはIDを口頭で伝える
        ↓
検証ページにアクセス（下記URL）
        ↓
認定IDを入力して「検証する」をクリック
        ↓
結果を確認
```

### 2.3 検証ページURL

> **https://sites.google.com/dendai.ac.jp/kyoiku-ninsho/verify**（準備中）
>
> ※ スマートフォンからもアクセス可能です

### 2.4 検証結果の見方

| 表示 | 意味 | 採用可否 |
|------|------|:--------:|
| ✅ **認定有効** | 研修修了済み、資格あり | ○ |
| ⚠️ **資格停止中** | 規律違反により一時停止 | × |
| ❌ **資格取消済み** | 重大な違反により永久取消 | × |
| ⏰ **有効期限切れ** | 認定から1年以上経過 | × |
| ❓ **IDが見つかりません** | 入力ミスまたは偽造の疑い | 要確認 |

> [!IMPORTANT] 採用前に必ず確認を
>
> - **「認定有効」以外の学生は採用しないでください**
> - 本プログラムの求人に応募できるのは認定学生のみです
> - 確認せずに採用した場合、トラブル時に大学のサポートを受けられない場合があります

### 2.5 確認できない場合

| 状況 | 対処法 |
|------|--------|
| 学生がIDを忘れた | 合格メールを探すよう依頼（大学メールに送信済み） |
| IDが見つからない | 学生に大学で再発行を依頼するよう伝える |
| システムエラー | 時間をおいて再度お試しください |

## 3. トラブル報告フォームの使い方

大学への連絡は、**電話ではなく報告フォーム**でお願いしています。

### 3.1 報告フォームURL

> **https://forms.gle/xxxxx**（準備中）
>
> ※ QRコードは[43_緊急連絡先カード_雛形](43_緊急連絡先カード_雛形.md)をご覧ください。

### 3.2 報告フォームの入力項目

| 項目 | 内容 |
|---|---|
| 報告者情報 | 企業名・担当者名・連絡先 |
| 対象学生 | 学籍番号または氏名 |
| 報告種別 | 選択式（無断欠勤 / 勤務態度不良 / 規律違反 / 労災報告 / その他） |
| 状況の詳細 | 具体的な事象・日時・経緯（自由記述） |
| 希望する対応 | 選択式（認定取消の検討 / 大学から学生への注意 / 報告のみ） |

### 3.3 報告後の流れ

1. **自動返信メール**が届きます（「受理しました」の確認）
2. 事務局が**週1回**まとめて確認・対応します
3. 対応結果は**メールで通知**します

> [!NOTE]
> 緊急対応（即日対応）は行っておりません。緊急の場合は警察・消防など適切な機関にご連絡ください。

## 4. 学生への指導のコツ

認定研修を受けた学生でも、実務は初めてです。以下のポイントを参考に、育成にご協力ください。

### 4.1 指示の出し方

| ポイント | 説明 |
|---|---|
| **具体的に** | 「ちゃんとやって」ではなく「〇〇を△個、□時までに」 |
| **メモを取らせる** | 「メモして」と声をかける。学生は研修で習っています |
| **復唱させる** | 「今の指示、復唱してみて」で理解度を確認 |
| **質問しやすい雰囲気** | 「わからなかったら聞いて」と最初に伝える |

### 4.2 叱り方

| ポイント | 説明 |
|---|---|
| **その場で** | 時間が経つと何が悪かったか伝わらない |
| **事実ベースで** | 「〇〇が間違っていた」と具体的に |
| **改善策とセットで** | 「次からは△△してね」と伝える |
| **人格否定しない** | 「ダメな奴」ではなく「この作業がダメ」 |

### 4.3 期待しすぎない

- 学生は「プロの卵」であり、即戦力ではありません
- 最初の1ヶ月は「教育投資」と考えてください
- 3ヶ月後に戦力化することを目標に

## 5. よくあるケースと対処法

### Q1. 学生が試験期間に休みたいと言ってきた

**A.** 試験期間中の休暇取得は、利用規約（第5条2項）で企業に配慮義務があります。可能な限りシフト調整をお願いします。

### Q2. 学生が突然辞めたいと言ってきた

**A.** 雇用契約は企業と学生の間で締結されています。学生から退職の相談があった場合は、話し合いのうえ対応してください。大学への報告は不要です。

### Q3. 学生のSNS投稿で社内の写真が公開されていた

**A.** 機密漏洩に該当する可能性があります。証拠を保全し、トラブル報告フォームでご連絡ください。認定取消を検討します。

### Q4. 学生が遅刻を繰り返している

**A.** まずは企業内で注意・指導をお願いします。改善しない場合は、トラブル報告フォームでご連絡ください。

### Q5. 採用したいが正社員として雇えるか

**A.** 大歓迎です。本プログラムは学生のキャリア形成を目的としています。卒業後の正社員採用に制限はありません。
