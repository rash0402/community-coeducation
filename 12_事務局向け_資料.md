# 事務局説明資料：工学部第二部 地域共育プログラム導入について

発表者： 工学部第二部長 五十嵐 洋

目的： 事務局の業務負担増と法的リスクを完全に排除した新モデルの承認を得ること。

## 1. 提案の核心：なぜモデルを変更したか

### 1.1 従来の懸念点と新たな定義

|   |   |   |
|---|---|---|
|**項目**|**従来の斡旋モデル（負担大）**|**新たなプラットフォームモデル（負担極小）**|
|**大学の役割**|仲介・マッチング・トラブル介入|**情報の掲示** と **学生の質保証（認定）**|
|**事務局工数**|無限大（個別調整・電話対応・クレーム処理）|**週30分程度**（データチェック・URL公開）|
|**法的リスク**|紹介責任・賠償連帯責任の懸念|免責規約により、**法的責任を企業と学生に限定**|
|**募集形態**|大学職員による営業|企業がWebフォームから**自主登録（セルフサービス）**|

> **結論:** 事務局が「人」として動くのをやめ、**「システム（場）」**に徹することで、プロジェクトの**持続可能性（サステナビリティ）**を担保します。

### 1.2 事務負担をゼロにするためのコア施策

以下の施策により、職員の**「電話対応」と「窓口対応」を物理的にゼロ**にします。

1. **デジタル認定証（窓口廃止）:**
    
    - 学生証へのシール貼付をやめ、研修合格後に自動送信される**「合格メール」を認定証**とします。学生は窓口に来る必要がなく、職員の対面対応が消滅します。
        
2. **自動リスト化（手作業廃止）:**
    
    - 求人票のPDF化、印刷、貼り付け作業を廃止。企業フォームの回答を**スプレッドシートの閲覧専用URL**で公開します。職員の作業は「NG求人の行を非表示にする」操作のみです。
        
3. **学生スタッフの活用（オプション）:**
    
    - 将来的には、週次のデータチェックや掲示板の更新といったルーチン作業を、**学生スタッフに委譲（RA/TA）**し、職員の実働時間をゼロに近づけます。
        

## 2. リスク管理と法的防衛ライン

最も懸念される「法的リスク」と「学生の安全」は、規約とマニュアルで完全にカバーされています。

### 2.1 法的リスクの遮断（利用規約の効力）

企業向け利用規約（第9条・第5条）により、以下の問題を大学から切り離します。

|   |   |   |
|---|---|---|
|**リスク**|**対策（規約上の根拠）**|**事務局の対応**|
|**労働災害**|**労災保険の適用を企業に義務化**（第5条4項）。大学の学研災は適用外。|Lv.3として記録し、学生相談室へ誘導。大学は一切責任を負わない。|
|**損害賠償**|業務上の損害は**原則企業が責任を負う**（第5条5項）。|「規約の通り、当事者間で解決してください」と回答し、介入しない。|
|**紹介責任**|本件は「職業紹介ではない（非仲介）」（第2条2項）。|「マッチング保証ではない」と回答し、応募がないことへの苦情を拒否。|

### 2.2 トラブル対応の分散管理（トリアージ）

事務局への電話対応を排除するため、トラブルの深刻度に応じて窓口を明確に分けます。

|   |   |   |   |
|---|---|---|---|
|**レベル**|**内容**|**担当窓口**|**事務局の対応**|
|**Lv.1**|遅刻・ミス|企業 ⇔ 学生（当事者間）|**介入禁止。** 企業には「就業規則で対応を」と案内。|
|**Lv.2**|継続的な態度不良|Web報告フォーム|**システム対応。** 学生を次回以降の認定資格から除外。|
|**Lv.3**|ハラスメント・労災|学生相談室|**即時掲載停止処分**（企業出禁）。専門部署と連携。|

## 3. フィルタリング（不適格者の排除）

「質の悪い学生」と「不適切な企業」は、システムと研修で排除します。

### 3.1 企業側のフィルタリング（求人登録フォーム）

フォームの設問と分岐ロジックにより、事務局のチェック前にNG企業を自動的に弾きます。

- **移動時間チェック:** 60分超を申告した企業は**自動的に登録不可画面へ誘導**。
    
- **必須遵守事項:** 「労災保険適用」「試験期間の配慮」に「いいえ」と回答した場合、**登録不可画面へ誘導**。
    
- **実在性チェック:** 初回登録時は**法人番号**を確認し、実在しない企業は掲載しない。
    

### 3.2 学生側のフィルタリング（認定研修）

この研修は、覚悟のない学生を自発的に辞退させるための「参入障壁」として機能します。

- **安全衛生:** 5S/KYTの基礎知識を必須とし、**現場での事故リスクを低減**。
    
- **コンプライアンス:** **SNSへの機密情報投稿が訴訟対象**となることを明記し、倫理観のない学生を排除。
    
- **自己誓約:** 「学業不振時は辞退する」と学生自身に誓約させ、**学業優先の責任を学生個人に負わせる**。
    

## 4. 求められる協力と次のステップ

### 4.1 事務局（地域連携担当）への依頼

貴部署にお願いする作業は、このマニュアルと運用ルールに基づき、**「週に一度、画面を見て、マニュアル通りの定型作業を行うこと」**のみです。

- **作業の具体例:** NG求人の行をスプレッドシートで非表示にする（数分）。
    
- **特典:** このプロジェクトの実績は、貴部署の**「地域貢献・産学連携の実績」**としてすべて報告されます。
    

### 4.2 次のステップ

すべての準備が整いました。あとは実制作に入ります。

- **広報ツールの制作:** **「第一期モデル企業への依頼状（協力要請）」**の作成。
    
    - 商工会議所での一般公募の前に、信頼できる企業にアプローチし、コールドスタート問題を回避します。
